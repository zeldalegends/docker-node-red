# node-red
### Ambiente Node-RED vuoto con dashboard
    git clone https://github.com/zeldalegends/node-red.git
    cd node-red
    docker build .
### Ambiente Node-RED vuoto con dashboard e TIG stack
    git clone https://github.com/zeldalegends/node-red.git
    cd node-red
    docker-compose up
### Ambiente Node-RED completo (PINK demo site)
    git clone https://github.com/zeldalegends/node-red.git
    cd node-red
    sh run.sh
